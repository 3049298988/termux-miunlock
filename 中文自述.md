# Termux MiUnlock

可用于检索小米设备的引导加载程序解锁令牌的程序。（并解锁引导加载程序）使用 Termux

**注意：此工具无法绕过 7、14、30 天的解锁时间。**

• [如何使用 Termux 解锁小米设备引导加载程序？](https://youtu.be/zTArE7sixJI?si=p8m0iQ-D3_abJ_23)


## 用法
```console
平台：使用 Termux。
*     DATA                存储库安装 account.apk，登录并粘贴令牌
      --debug             输出该工具的debug调试信息
      --help              显示帮助消息
*     --product=PRODUCT   用于验证设备代号
      --region=REGION     解锁服务器主机所在区域:
                          india, global, china,
                          russia, europe
                          默认: india
  
*     --token=TOKEN       用于验证的设备令牌
      --version           查看工具版本
```


## 基本要求
1) 已绑定的小米账户
2) 两个Android设备（主机和从机）
3) USB-OTG或者是C2C数据线
4) 互联网连接


## 操作说明
1) 在主机上安装所需软件或环境 [termux](https://github.com/termux/termux-app), [termux-api](https://github.com/termux/termux-api) 和 ```account.apk```

2) 在目标设备上登录并绑定您的小米账号

3) 用下面的命令克隆此存储库。
```console
git clone https://github.com/RohitVerma882/termux-miunlock.git && cd termux-miunlock
```

4) 运行 ```setup.sh``` 以安装所需的软件包。
```console
chmod +x setup.sh && ./setup.sh
```

5) 获取设备的 ```product```
```console
mi-fastboot getvar product
```

6) 获取设备的 ```token```
```console
mi-fastboot getvar token
```

7) 获取MTK设备的 ```token```
```console
mi-fastboot oem get_token
```
如果您收到 2 或 3 个令牌，则合并它，
例如：
```console
// 合并前 
(bootloader) token: VQECMAEQTSdjm281zqPylolzfxy3bQMGbWVy
(bootloader) token: bGluAhTVfQBXJGUJ78qoZQ0ctBDLQ1PkJg==

// 合并后
VQECMAEQTSdjm281zqPylolzfxy3bQMGbWVybGluAhTVfQBXJGUJ78qoZQ0ctBDLQ1PkJg==
```

8) 使用必需的参数运行 ```get_token.sh``` 
```console
chmod +x get_token.sh
```
```console
./get_token.sh --region=global --product=PRODUCT --token=TOKEN DATA
```
如果运行成功，它将为您提供一个非常长的字符串，即解锁令牌。

如果出现错误 20045，则应设置您在 account.apk 中查询到的服务器区域

可用选项: `india, global, china, russia, europe`

```console
./get_token.sh --region=REGION --product=PRODUCT --token=TOKEN DATA
```

9) 将解锁令牌字符串转换为二进制令牌。
  ```console
  echo "UNLOCK_TOKEN" | xxd -r -p > token.bin
  ```

10) 解锁方法:
```console
mi-fastboot stage token.bin && mi-fastboot oem unlock
```
或者 (跳过步骤9):
```console
mi-fastboot oem-unlock "UNLOCK_TOKEN"
```


设备将恢复出厂设置并成功解锁BL。
